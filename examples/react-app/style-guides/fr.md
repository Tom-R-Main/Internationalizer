# French (fr) — Style Guide

## Basics
- **Locale:** `fr`
- **Script:** Latin, LTR
- **CLDR plural forms:** one, many, other
- **Text expansion:** +15–25% vs English

## Formality
Use **vous** (formal "you") for all UI text. This is a professional software application.

## Punctuation
- Non-breaking space before `:`, `;`, `!`, `?`
- Guillemets `« »` for quotation marks (with non-breaking spaces inside)
- Decimal separator: comma (1 234,56)
- Thousands separator: non-breaking space (1 000 000)

## Grammar
- Use masculine as grammatical default when gender is unknown
- Prefer active voice over passive
- Keep sentences short and direct
