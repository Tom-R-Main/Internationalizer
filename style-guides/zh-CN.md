# Simplified Chinese (zh-CN) — Translation Style Guide

## Language Profile
- Locale: `zh-CN`
- Script: CJK, LTR
- CLDR plural forms: other
- Text expansion vs English: -30% 到 -50% (中文字符数通常少于英文，但全角字符视觉宽度较大)

## Tone & Formality
在软件界面和文档中，语气应保持专业、清晰、友好且客观。
- **正式度**：使用现代标准汉语，避免过于口语化、网络用语或方言。在需要称呼用户时，使用尊称“您”而不是“你”。但在 UI 文本（如按钮、菜单）中应尽量省略主语，以保持界面简洁。
- **语态**：采用直接、主动的语态。避免生硬的机器翻译腔，确保符合中文母语者的阅读习惯。
- **错误提示**：应具有建设性，直接告知用户发生了什么以及如何解决。避免指责用户（例如，使用“密码不正确”而非“您输入了错误的密码”）。

## Grammar
- **按钮与操作（Imperatives）**：使用简洁的动宾结构短语，直接表达动作。例如，使用“保存更改”而不是“去保存您的更改”。按钮文本中通常不使用标点符号。
- **标签与标题（Noun phrases）**：使用精炼的名词短语。例如，使用“账户设置”而不是“关于账户的设置”。
- **避免“翻译腔”**：
  1. **省略多余代词**：英文常说 "Save your profile"，中文只需“保存个人资料”，无需译为“保存您的个人资料”。
  2. **慎用被动语态**：英文的被动语态在中文中通常应转换为主动语态或无主语句。例如 "The file was deleted" 应译为“文件已删除”，而非“文件被删除了”。
  3. **避免滥用“进行”**：直接使用动词即可。例如，用“正在搜索...”代替“正在进行搜索...”。
- **中英文混排空格**：在英文字母/数字与中文字符之间必须保留一个半角空格（例如：“支持 iOS 15 及以上版本”），以提升排版美观度和可读性。

## Pluralization
中文没有严格意义上的名词复数形态变化。CLDR 复数规则仅包含 `other` 这一种类别。
- 无论数字大小（如 0, 1, 2, 100），名词本身不发生形态变化。
- 翻译包含变量的复数句子时，只需直接将数字变量与量词和名词结合。例如："1 file" 译为“1 个文件”，"3 files" 译为“3 个文件”。
- 遇到英文的 "(s)" 形式（如 "item(s)"），在中文里通常直接翻译为“项目”，不需要刻意翻译出复数含义，除非语境必须强调多个（此时可使用“多个项目”）。

## Punctuation & Typography
- **标点符号**：中文语境下必须使用全角标点符号（如 `，` `。` `！` `？` `：`）。
- **引号**：中国大陆地区默认使用全角双引号 `“` 和 `”`，内部嵌套时使用单引号 `‘` 和 `’`。
- **数字与分隔符**：小数点使用半角句号 `.`（如 3.14），千位分隔符使用半角逗号 `,`（如 1,000）。
- **日期格式**：推荐使用 `YYYY-MM-DD`（如 2023-10-25）或 `YYYY年MM月DD日`。严禁使用具有歧义的 `MM/DD/YYYY`。
- **时间格式**：推荐使用 24 小时制（如 14:30）。若必须使用 12 小时制，需加上明确的中文标识（如“下午 2:30”）。

## Terminology
| English | Simplified Chinese | Notes |
|---------|--------------------|-------|
| Save | 保存 | 用于保存当前进度或设置的按钮。 |
| Cancel | 取消 | 用于中止当前操作，关闭对话框。 |
| Delete | 删除 | 永久移除数据或文件。 |
| Settings | 设置 | 应用程序的配置或偏好选项。 |
| Search | 搜索 | 查找内容的输入框或按钮。 |
| Error | 错误 | 系统故障或操作失败的提示。 |
| Loading | 加载中 | 界面正在获取数据时的状态提示，通常带省略号（加载中...）。 |
| Dashboard | 仪表盘 | 系统的概览主页或数据看板。 |
| Notifications | 通知 | 系统发出的消息、警告或提醒。 |
| Sign in | 登录 | 用户进入系统的操作。注意必须是“登录”，严禁写成“登陆”。 |
| Sign out | 退出登录 | 用户离开系统的操作。也可简称为“退出”。 |
| Submit | 提交 | 发送表单或数据的操作。 |
| Profile | 个人资料 | 用户的个人信息页面或设置。 |
| Help | 帮助 | 提供支持文档或指南的入口。 |
| Close | 关闭 | 隐藏弹窗或退出当前界面的操作。 |
