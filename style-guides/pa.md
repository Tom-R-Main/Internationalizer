# Punjabi (pa) — Translation Style Guide

## Language Profile
- Locale: `pa`
- Script: Gurmukhi, LTR
- CLDR plural forms: one, other
- Text expansion vs English: 15-25%

## Tone & Formality
ਸਾਫਟਵੇਅਰ ਅਤੇ ਐਪਲੀਕੇਸ਼ਨਾਂ ਲਈ ਭਾਸ਼ਾ ਦਾ ਲਹਿਜਾ (tone) ਪੇਸ਼ੇਵਰ, ਸਪਸ਼ਟ ਅਤੇ ਸਤਿਕਾਰਯੋਗ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ। ਵਰਤੋਂਕਾਰ ਨੂੰ ਸੰਬੋਧਨ ਕਰਨ ਲਈ ਹਮੇਸ਼ਾ ਆਦਰਸੂਚਕ ਪੜਨਾਂਵ "ਤੁਸੀਂ" ਅਤੇ ਇਸਦੇ ਰੂਪਾਂ (ਤੁਹਾਡਾ, ਤੁਹਾਨੂੰ) ਦੀ ਵਰਤੋਂ ਕਰੋ; ਗੈਰ-ਰਸਮੀ "ਤੂੰ" ਜਾਂ "ਤੇਰਾ" ਦੀ ਵਰਤੋਂ ਕਦੇ ਨਾ ਕਰੋ। ਭਾਸ਼ਾ ਸਿੱਧੀ ਅਤੇ ਵਰਤੋਂਕਾਰ-ਪੱਖੀ (user-friendly) ਰੱਖੋ। ਬਹੁਤ ਜ਼ਿਆਦਾ ਔਖੇ, ਪੁਰਾਣੇ ਜਾਂ ਸਾਹਿਤਕ ਸ਼ਬਦਾਂ ਦੀ ਬਜਾਏ ਆਮ ਬੋਲਚਾਲ ਵਾਲੇ ਪਰ ਮਿਆਰੀ ਪੰਜਾਬੀ ਸ਼ਬਦਾਂ ਨੂੰ ਤਰਜੀਹ ਦਿਓ।

## Grammar
1. **ਬਟਨਾਂ ਲਈ ਆਦੇਸ਼ਾਤਮਕ ਰੂਪ (Imperatives):** ਬਟਨਾਂ, ਲਿੰਕਾਂ ਅਤੇ ਕਮਾਂਡਾਂ ਲਈ ਹਮੇਸ਼ਾ ਆਦਰਸੂਚਕ ਹੁਕਮੀ ਰੂਪ ਵਰਤੋ (ਜਿਵੇਂ ਕਿ "ਕਰੋ")। ਉਦਾਹਰਨ: 'Save' ਲਈ "ਸੇਵ ਕਰੋ" ਜਾਂ "ਸੰਭਾਲੋ" ਵਰਤੋ, ਨਾ ਕਿ "ਸੇਵ ਕਰ" ਜਾਂ "ਸੰਭਾਲ"।
2. **ਵਾਕ ਬਣਤਰ (Sentence Structure):** ਪੰਜਾਬੀ ਵਿੱਚ ਵਾਕ ਦੀ ਬਣਤਰ ਕਰਤਾ-ਕਰਮ-ਕਿਰਿਆ (SOV - Subject Object Verb) ਹੁੰਦੀ ਹੈ। ਅੰਗਰੇਜ਼ੀ ਦੀ ਬਣਤਰ (SVO) ਦੀ ਸਿੱਧੀ ਨਕਲ ਨਾ ਕਰੋ। ਕਿਰਿਆ ਹਮੇਸ਼ਾ ਵਾਕ ਦੇ ਅੰਤ ਵਿੱਚ ਆਉਣੀ ਚਾਹੀਦੀ ਹੈ।
3. **ਤਕਨੀਕੀ ਸ਼ਬਦਾਵਲੀ (Tech terms):** ਜੇਕਰ ਕਿਸੇ ਤਕਨੀਕੀ ਸ਼ਬਦ ਦਾ ਠੇਠ ਪੰਜਾਬੀ ਅਨੁਵਾਦ ਬਹੁਤ ਗੁੰਝਲਦਾਰ ਜਾਂ ਅਜੀਬ ਲੱਗਦਾ ਹੈ, ਤਾਂ ਉਸਨੂੰ ਗੁਰਮੁਖੀ ਵਿੱਚ ਲਿਪੀਅੰਤਰਨ (transliterate) ਕਰਨਾ ਬਿਹਤਰ ਹੈ (ਜਿਵੇਂ ਕਿ File ਲਈ "ਫਾਈਲ", App ਲਈ "ਐਪ")।
4. **ਗਲਤੀ ਸੁਨੇਹੇ (Error Messages):** ਗਲਤੀ ਸੁਨੇਹਿਆਂ ਨੂੰ ਸਿੱਧਾ ਅਤੇ ਨਿਰਪੱਖ ਰੱਖੋ। ਉਦਾਹਰਨ: "You entered the wrong password" ਦੀ ਬਜਾਏ ਕਰਮਵਾਚ (Passive voice) ਵਿੱਚ "ਗਲਤ ਪਾਸਵਰਡ ਦਾਖਲ ਕੀਤਾ ਗਿਆ ਹੈ" ਵਰਤਣਾ ਵਧੇਰੇ ਪੇਸ਼ੇਵਰ ਹੈ, ਤਾਂ ਜੋ ਵਰਤੋਂਕਾਰ 'ਤੇ ਸਿੱਧਾ ਦੋਸ਼ ਨਾ ਲੱਗੇ।

## Pluralization
ਪੰਜਾਬੀ ਵਿੱਚ CLDR ਦੇ ਦੋ ਬਹੁ-ਵਚਨ ਰੂਪ ਹਨ: 'one' (ਇੱਕ) ਅਤੇ 'other' (ਹੋਰ)।
- **One (ਇੱਕ):** ਜਦੋਂ ਗਿਣਤੀ 1 ਹੋਵੇ। ਉਦਾਹਰਨ: "1 ਫਾਈਲ ਮਿਟਾਈ ਗਈ" (1 file deleted)।
- **Other (ਹੋਰ):** ਜਦੋਂ ਗਿਣਤੀ 0, 2, ਜਾਂ ਇਸ ਤੋਂ ਵੱਧ ਹੋਵੇ। ਉਦਾਹਰਨ: "0 ਫਾਈਲਾਂ ਮਿਟਾਈਆਂ ਗਈਆਂ", "3 ਫਾਈਲਾਂ ਮਿਟਾਈਆਂ ਗਈਆਂ" (3 files deleted)।
ਧਿਆਨ ਦਿਓ ਕਿ ਬਹੁ-ਵਚਨ ਬਣਾਉਣ ਵੇਲੇ ਨਾਂਵ (ਫਾਈਲ -> ਫਾਈਲਾਂ) ਦੇ ਨਾਲ-ਨਾਲ ਕਿਰਿਆ ਦਾ ਰੂਪ ਵੀ ਬਦਲਦਾ ਹੈ (ਜਿਵੇਂ 'ਗਈ' ਤੋਂ 'ਗਈਆਂ')।

## Punctuation & Typography
- **ਪੂਰਨ ਵਿਰਾਮ (Full Stop):** ਪੰਜਾਬੀ ਵਿੱਚ ਵਾਕ ਦੇ ਅੰਤ ਵਿੱਚ ਡੰਡੀ (।) ਦੀ ਵਰਤੋਂ ਕੀਤੀ ਜਾਂਦੀ ਹੈ, ਅੰਗਰੇਜ਼ੀ ਵਾਲੇ ਬਿੰਦੂ (.) ਦੀ ਨਹੀਂ।
- **ਹਵਾਲਾ ਚਿੰਨ੍ਹ (Quotes):** ਡਬਲ ਹਵਾਲਾ ਚਿੰਨ੍ਹ ("...") ਦੀ ਵਰਤੋਂ ਕਰੋ।
- **ਅੰਕ ਅਤੇ ਗਣਿਤ (Numbers):** ਦਸ਼ਮਲਵ ਲਈ ਬਿੰਦੂ (10.50) ਅਤੇ ਹਜ਼ਾਰਾਂ ਲਈ ਕੌਮਾ (1,000) ਵਰਤੋ। ਅੰਕਾਂ ਲਈ ਅੰਗਰੇਜ਼ੀ (0-9) ਅੰਕਾਂ ਦੀ ਹੀ ਵਰਤੋਂ ਕਰੋ, ਗੁਰਮੁਖੀ ਅੰਕਾਂ (੦-੯) ਦੀ ਵਰਤੋਂ ਸਾਫਟਵੇਅਰ ਵਿੱਚ ਆਮ ਤੌਰ 'ਤੇ ਨਹੀਂ ਕੀਤੀ ਜਾਂਦੀ।
- **ਤਾਰੀਖ (Dates):** DD/MM/YYYY ਫਾਰਮੈਟ ਦੀ ਵਰਤੋਂ ਕਰੋ (ਜਿਵੇਂ ਕਿ 31/12/2023)।
- **ਸਮਾਂ (Time):** ਆਮ ਤੌਰ 'ਤੇ 12-ਘੰਟੇ ਵਾਲਾ ਫਾਰਮੈਟ ਵਰਤਿਆ ਜਾਂਦਾ ਹੈ, ਜਿਸ ਨਾਲ 'ਸਵੇਰੇ' (AM) ਜਾਂ 'ਸ਼ਾਮ/ਰਾਤ' (PM) ਲਗਾਇਆ ਜਾਂਦਾ ਹੈ।

|---------|---------|-------|
| Save | ਸੇਵ ਕਰੋ | 'ਸੰਭਾਲੋ' ਵੀ ਵਰਤਿਆ ਜਾ ਸਕਦਾ ਹੈ, ਪਰ UI ਵਿੱਚ 'ਸੇਵ ਕਰੋ' ਵਧੇਰੇ ਆਮ ਅਤੇ ਸਪਸ਼ਟ ਹੈ। |
| Cancel | ਰੱਦ ਕਰੋ | ਕਿਸੇ ਕਾਰਵਾਈ ਜਾਂ ਪ੍ਰਕਿਰਿਆ ਨੂੰ ਰੋਕਣ ਲਈ। |
| Delete | ਮਿਟਾਓ | 'ਡਿਲੀਟ ਕਰੋ' ਵੀ ਪ੍ਰਵਾਨਿਤ ਹੈ, ਪਰ 'ਮਿਟਾਓ' ਵਧੀਆ ਪੰਜਾਬੀ ਸ਼ਬਦ ਹੈ। |
| Settings | ਸੈਟਿੰਗਾਂ | ਹਮੇਸ਼ਾ ਬਹੁ-ਵਚਨ ਰੂਪ ਵਿੱਚ ਵਰਤੋ। |
| Search | ਖੋਜੋ | ਬਟਨ ਜਾਂ ਕਮਾਂਡ ਲਈ 'ਖੋਜੋ' ਵਰਤੋ। ਜੇਕਰ ਨਾਂਵ (Noun) ਵਜੋਂ ਹੋਵੇ ਤਾਂ 'ਖੋਜ' ਵਰਤੋ। |
| Error | ਗਲਤੀ | ਕਿਸੇ ਤਕਨੀਕੀ ਖਰਾਬੀ ਜਾਂ ਤਰੁੱਟੀ ਲਈ। |
| Loading | ਲੋਡ ਹੋ ਰਿਹਾ
