# Korean (ko) — Translation Style Guide

## Language Profile
- Locale: `ko`
- Script: Hangul, LTR
- CLDR plural forms: other
- Text expansion vs English: -10% ~ -15% (일반적으로 영문보다 텍스트 길이가 짧아짐)

## Tone & Formality
소프트웨어 UI 및 문서는 전문적이고 명확하며 예의 바른 어조를 유지해야 합니다.
- **일반 문장 및 안내문:** '하십시오'체(격식체, -습니다/-비다)를 기본으로 사용하여 사용자에게 신뢰감을 줍니다. 서비스 대상에 따라 친근한 느낌을 주어야 하는 B2C 앱의 경우 '해요'체(비격식체, -아/어요)를 일관되게 사용할 수 있습니다.
- **UI 버튼 및 메뉴:** 공간 절약과 명확성을 위해 명사형 종결(예: '저장', '삭제') 또는 '-기' 형태(예: '더 보기', '보내기')를 사용합니다.
- **어조:** 사용자를 탓하지 않는 긍정적이고 능동적인 표현을 사용합니다. (예: "잘못된 입력입니다" 대신 "올바른 값을 입력해 주세요" 사용)

## Grammar
- **대명사 생략 (Drop Pronouns):** 영어의 'you', 'your', 'please'를 한국어로 직역하면 매우 어색합니다. '당신', '당신의', '부디'와 같은 표현은 생략하고 문맥에 맞게 자연스럽게 번역합니다. (예: "Enter your password" -> "비밀번호를 입력하세요")
- **명령문 (Imperatives):** 강압적인 명령조('-해라')는 피하고, 정중한 요청 형태('-하세요', '-해 주세요')를 사용합니다. 버튼이나 짧은 레이블에서는 명사형으로 대체합니다.
- **수동태 피하기 (Avoid Passive Voice):** 한국어는 능동태가 훨씬 자연스럽습니다. 사물을 주어로 하는 영어의 수동태 문장은 사용자를 주어로 하거나 자연스러운 능동태로 변환합니다. (예: "The file was successfully deleted" -> "파일을 성공적으로 삭제했습니다" 또는 "파일이 삭제되었습니다")
- **명사구 나열 (Noun Phrases):** UI 레이블이나 메뉴명은 간결한 명사구로 번역합니다. 불필요한 조사('의', '에 대한')는 생략하는 것이 가독성에 좋습니다. (예: "Settings for user profile" -> "사용자 프로필 설정")

## Pluralization
한국어는 CLDR 복수형 규칙에서 `other` 단일 범주만 사용합니다. 영어와 달리 복수형 접미사('-들')를 엄격하게 사용하지 않으며, 특히 숫자 뒤에 단위 명사가 올 때는 단수형을 쓰는 것이 훨씬 자연스럽습니다. 숫자 변수가 포함된 문장에서는 복수형 표기에 얽매이지 말고 문맥에 맞게 번역하세요.
- **예시:** "1 file" -> "파일 1개"
- **예시:** "3 files" -> "파일 3개" (O) / "파일들 3개" (X) / "3개의 파일들" (X)

## Punctuation & Typography
- **따옴표:** UI 요소를 지칭하거나 강조할 때는 작은따옴표(' ')를, 직접 인용하거나 대화문을 표기할 때는 큰따옴표(" ")를 사용합니다.
- **숫자 기호:** 천 단위 구분 기호는 쉼표(,)를, 소수점은 마침표(.)를 사용합니다. (예: 1,234.56)
- **날짜 형식:** `YYYY년 MM월 DD일` 또는 `YYYY-MM-DD` 형식을 사용합니다. (예: 2023년 10월 24일)
- **시간 형식:** 24시간제(예: 14:30) 또는 12시간제(예: 오후 2:30)를 사용합니다. 'AM/PM'은 '오전/오후'로 번역하여 시간 앞에 배치합니다.
- **띄어쓰기:** 한글과 영문/숫자가 혼용될 때는 가독성을 위해 그 사이에 공백을 하나 두는 것을 권장합니다. 단, 숫자와 단위 명사 사이는 붙여 쓰는 것을 허용합니다. (예: "총 3GB의 데이터")

## Terminology
| English | Korean | Notes |
|---------|---------|-------|
| Save | 저장 | 데이터를 보존하고 기록할 때 사용합니다. |
| Cancel | 취소 | 진행 중인 작업이나 변경 사항을 중단할 때 사용합니다. |
| Delete | 삭제 | 데이터나 파일을 영구적으로 지울 때 사용합니다. |
| Settings | 설정 | 앱이나 시스템의 환경을 변경하는 메뉴입니다. |
| Search | 검색 | 특정
