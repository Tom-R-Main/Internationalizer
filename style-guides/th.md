# Thai (th) — Translation Style Guide

## Language Profile
- Locale: `th`
- Script: Thai, LTR
- CLDR plural forms: other
- Text expansion vs English: 15-20%

## Tone & Formality
ใช้ระดับภาษาที่เป็นทางการ กึ่งทางการ และสุภาพ (Professional and polite) แต่อ่านเข้าใจง่ายและกระชับ หลีกเลี่ยงการใช้คำศัพท์เฉพาะทางที่ซับซ้อนหรือคำสแลง ไม่จำเป็นต้องใช้คำลงท้าย (ครับ/ค่ะ) ใน UI ทั่วไป เช่น ปุ่ม หรือเมนู เพื่อความกระชับ แต่สามารถใช้ในข้อความต้อนรับหรือข้อความแสดงข้อผิดพลาดที่ต้องการความนุ่มนวล สรรพนามแทนผู้ใช้ให้ใช้คำว่า "คุณ" และหลีกเลี่ยงการแปลสรรพนาม "I", "We", "My" หากไม่จำเป็น (เช่น "My Account" ให้แปลว่า "บัญชี" ไม่ใช่ "บัญชีของฉัน")

## Grammar
- **ปุ่มและคำสั่ง (Imperatives):** ใช้คำกริยาขึ้นต้นข้อความบนปุ่มโดยไม่ต้องมีคำว่า "โปรด" (Please) หรือ "กรุณา" ยกเว้นกรณีที่เป็นคำสั่งที่สำคัญมากหรือเป็นประโยคขอร้องแบบเต็ม เช่น ใช้ "บันทึก" (Save) ไม่ใช่ "โปรดบันทึก"
- **ป้ายกำกับ (Labels):** ใช้คำนามหรืออาการนาม (การ/ความ) สำหรับหัวข้อหรือป้ายกำกับ เช่น "การตั้งค่า" (Settings) หรือ "ความปลอดภัย" (Security)
- **การเว้นวรรค (Spacing):** ภาษาไทยไม่เว้นวรรคระหว่างคำ แต่ให้เว้นวรรคระหว่างประโยค วลีที่แยกจากกัน และ**ต้องเว้นวรรคหน้าและหลังตัวเลขหรือคำภาษาอังกฤษเสมอ** เมื่อเขียนปะปนกับภาษาไทย เช่น "ไฟล์ขนาด 5 MB ถูกลบแล้ว"
- **โครงสร้างประโยค:** ภาษาไทยใช้โครงสร้าง คำนาม + คำคุณศัพท์ (Noun + Adjective) เสมอ เช่น "New file" ต้องแปลว่า "ไฟล์ใหม่" ไม่ใช่ "ใหม่ไฟล์"

## Pluralization
ภาษาไทยไม่มีการเปลี่ยนรูปคำนามหรือกริยาเพื่อแสดงพหูพจน์ (ใช้หมวดหมู่ CLDR: `other` สำหรับทุกตัวเลข) การแสดงจำนวนจะใช้โครงสร้าง "คำนาม + ตัวเลข + ลักษณนาม" หรือ "ตัวเลข + ลักษณนาม" เสมอ ไม่ว่าจำนวนนั้นจะเป็น 0, 1 หรือมากกว่า 1 ก็ตาม เช่น:
- 0 items = 0 รายการ
- 1 file = 1 ไฟล์
- 5 users = ผู้ใช้ 5 คน (หรือ 5 ผู้ใช้ ในบริบทของ UI ที่ต้องการความสั้น)

## Punctuation & Typography
- **เครื่องหมายคำพูด:** ใช้เครื่องหมายอัญประกาศคู่ ("...") สำหรับเน้นคำหรือข้อความ
- **ตัวเลข:** ใช้เลขอารบิก (0-9) เสมอ ใช้จุด (.) สำหรับทศนิยม และจุลภาค (,) สำหรับหลักพัน เช่น 1,234.56
- **วันที่:** ใช้รูปแบบ วัน/เดือน/ปี (DD/MM/YYYY) เช่น 31/12/2023 หรือใช้ชื่อเดือนย่อ เช่น 31 ธ.ค. 2023 (ใช้ปี ค.ศ. เป็นหลักสำหรับซอฟต์แวร์สากล)
- **เวลา:** นิยมใช้รูปแบบ 24 ชั่วโมง (HH:MM) ตามด้วย "น." เช่น 14:30 น.
- **เครื่องหมายจบประโยค:** ภาษาไทยไม่ต้องใช้จุด Full stop (.) เมื่อจบประโยค

## Terminology

| English | Thai | Notes |
|---------|---------|-------|
| Save | บันทึก | ใช้สำหรับปุ่มบันทึกข้อมูลหรือการเปลี่ยนแปลง |
| Cancel | ยกเลิก | ใช้สำหรับปุ่มยกเลิกการกระทำหรือปิดหน้าต่างโดยไม่บันทึก |
| Delete | ลบ | ใช้สำหรับการลบข้อมูล ไฟล์ หรือบัญชี |
| Settings | การตั้งค่า | ใช้สำหรับเมนูหรือหน้าจอที่ให้ผู้ใช้ปรับแต่งระบบ |
| Search | ค้นหา | ใช้สำหรับช่องกรอกข้อความหรือปุ่มเพื่อค้นหาข้อมูล |
| Error | ข้อผิดพลาด | ใช้เป็นหัวข้อเมื่อระบบทำงานล้มเหลวหรือเกิดปัญหา |
| Loading | กำลังโหลด | ใช้แสดงสถานะขณะที่ระบบกำลังประมวลผลหรือดึงข้อมูล |
| Dashboard | แดชบอร์ด | ใช้ทับศัพท์สำหรับหน้ากระดานสรุปข้อมูล |
| Notifications | การแจ้งเตือน | ใช้สำหรับเมนูหรือหน้าต่างแสดงข้อความแจ้งเตือนระบบ |
| Sign in | เข้าสู่ระบบ | ใช้สำหรับการล็อกอินเข้าใช้งาน (Log in ก็ใช้คำนี้) |
| Sign out | ออกจากระบบ | ใช้สำหรับการล็อกเอาต์ (Log out ก็ใช้คำนี้) |
| Submit | ส่ง | ใช้สำหรับปุ่มส่งข้อมูลในแบบฟอร์ม |
| Profile | โปรไฟล์ | ใช้ทับศัพท์สำหรับหน้าข้อมูลส่วนตัวของผู้ใช้ |
| Help | ความช่วยเหลือ | ใช้สำหรับเมนูสนับสนุนหรือคู่มือการใช้งาน |
| Close | ปิด | ใช้สำหรับปุ่มปิดหน้าต่าง ป๊อปอัป หรือกล่องข้อความ |
